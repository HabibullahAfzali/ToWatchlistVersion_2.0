
# To-Watch List






To-Do app developed with Vue 3, Spring Boot and MySQL. It allows the user to organize a list of pending to watch movies and perform CRUD operations on them.

## Project Setup

```sh
npm install
```

```sh
npm install axios
```
## Recommended Setup

```sh
VSCode
Intellij
XAMPP
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```
